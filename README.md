## Overview

This project delivers actionable business intelligence from Northwind Traders' H1 2006 sales data.

The analysis covers the full pipeline — from raw SQL extraction to interactive Power BI dashboards — identifying top-performing products, revenue concentration risks, and employee performance patterns that can inform strategic decisions.

## Workflow

### 1. Data Collection
- Source: Northwind Traders H1 2006 sales database
- Raw data extracted via SQL queries and stored in `data/` for reproducibility

### 2. Data Cleaning & Transformation
- Standardized column names, data types, and formats
- Deduplicated records and ensured chronological integrity
- Aggregated data at monthly, product, and employee levels

### 3. Exploratory Analysis
- Computed KPIs: Total Revenue, Total Orders, Total Customers, Average Order Value
- Ranked top products, categories, and individual employee contributions
- Detected high revenue concentration and potential operational bottlenecks

### 4. Visualization
- Built interactive Power BI dashboards for business stakeholders
- Implemented a star schema model for clean, scalable data relationships

---

## Insights

- **Revenue concentration risk:** ~64% of total revenue is generated by only 3 products — a significant dependency that creates business fragility.
- **Category dominance:** Beverages and Seafood consistently lead across the period.
- **Employee performance gap:** Significant variation exists across the sales team; standardizing top performers' strategies could lift overall results.
- **Actionable recommendation:** Diversify product revenue sources and implement a best-practices playbook based on top employee behavior.

## How to Run

1. Place the raw Northwind H1 2006 SQL data in the `data/` folder
2. Run the extraction and transformation queries:
   ```bash
   mysql -u your_user -p northwind < sql/northwind_queries.sql
   ```
3. Open the Power BI dashboard:
   ```
   power_bi/northwind_dashboard.pbix
   ```
4. Explore KPIs, product rankings, and employee performance interactively
